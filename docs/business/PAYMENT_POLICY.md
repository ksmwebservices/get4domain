# Payment Policy — Get4Domain

## Implementation Payment

- **50% advance** — due before development begins (P001 kickoff).
- **50% final** — due before production deployment (P006 go-live).

Development does not start before the advance is received. Production
deployment does not proceed before the final payment is received.

## Hosting

- Managed hosting is billed **annually**, starting from the production
  go-live date.
- Hosting covers server, database, SSL, and routine platform maintenance
  (security patches, uptime monitoring) for the deployed application.

## Annual Maintenance Contract (AMC)

- Optional, contracted separately from hosting.
- Covers minor enhancements, bug fixes, and support requests outside the
  original scope, within agreed limits.
- Clients without an AMC are billed per change request for anything beyond
  hosting-level maintenance.

## Source Code

- Source code is **not delivered** to the client as part of the standard
  engagement.
- Source code delivery is only provided if separately contracted and paid
  for, documented in the SOW.

## Renewal

- Hosting renewal notices go out ahead of the annual due date.
- Non-renewal results in the environment being taken offline after the
  agreed grace period; data retention follows the standard backup policy
  before deletion.
