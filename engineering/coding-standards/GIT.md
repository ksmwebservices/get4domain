# Git Standards — Get4Domain v1.0

## Branches
main      -> Production (protected)
develop   -> Integration
feature/* -> New features
fix/*     -> Bug fixes
hotfix/*  -> Production emergency
release/* -> Release preparation

## Commit Format
<type>: <description>

Types: feat | fix | docs | chore | refactor | test | deploy

## Examples
feat: add vehicle availability calendar
fix: resolve GST rounding error
test: add driver assignment unit tests
deploy: configure nginx SSL
