# Client Onboarding Checklist — Get4Domain v1.0

## Pre-Development
- [ ] Requirement meeting done
- [ ] Proposal sent and approved
- [ ] Scope of Work finalized
- [ ] NDA signed
- [ ] Agreement signed
- [ ] 50% advance payment received
- [ ] Timeline approved

## Development Start
- [ ] GitHub repo created
- [ ] Dev subdomain configured (client-dev.get4domain.com)
- [ ] Project folder created
- [ ] Kickoff message sent to client

## Delivery
- [ ] Dev URL shared with client
- [ ] Client review period (5-7 days)
- [ ] Feedback incorporated
- [ ] Final approval in writing
- [ ] Final 50% payment received
- [ ] Production deployment done
- [ ] Production URL confirmed
- [ ] Admin training completed
- [ ] Handover document sent
